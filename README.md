Welcome to the **Adarsh 3D Interactive Portfolio**, a cutting-edge web experience designed to showcase the skills, projects, and journey of a passionate Full Stack and Django Backend Developer.

This project is not a standard scrolling website. It is a **cinematic, 3D scroll-driven journey** across a stylized, abstract map of India. As you scroll, the camera flies dynamically from the Northern mountains down to the Eastern bridges, transitioning through breathtaking lighting and atmospheric fog effects.

## 🚀 Tech Stack

This project was built with performance, aesthetics, and modern web standards in mind:

- **React.js & Next.js:** The core foundation for a robust, SEO-friendly, and blazing-fast web application.
- **Three.js & React Three Fiber:** Powering the stunning, interactive 3D map, geometries, and spatial environment.
- **GSAP (GreenSock) & ScrollTrigger:** Driving the buttery-smooth cinematic camera fly-throughs, dynamic lighting shifts, and element animations locked to your scroll.
- **Tailwind CSS:** Crafting the sleek, responsive, and glassmorphism-styled UI components that beautifully float over the 3D canvas.

## ✨ Key Features

- **Cinematic Scroll Journey:** The user's scroll directly controls the 3D camera, flying them across different zones (North, Delhi, UP Hub, East).
- **Dynamic Lighting & Fog:** As the camera enters new zones, the entire environment smoothly morphs—shifting from sky-blue winter fog to vibrant greens, majestic golden-hour hues, and twilight blues.
- **Floating 3D UI Overlays:** Resumes, education details, skills, and featured projects seamlessly fade into view as gorgeous, glass-like HTML overlays anchored to 3D landmarks.
- **Optimized Performance:** Carefully structured to maintain a smooth **60fps** experience across devices.

## 💻 Running the Project Locally

To run this beautifully crafted portfolio on your own machine, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/AdarshPathak9628/Adarsh.git
   ```

2. **Navigate to the project folder:**
   ```bash
   cd Adarsh
   ```

3. **Install the dependencies:**
   ```bash
   npm install --legacy-peer-deps
   ```
   *(We use `--legacy-peer-deps` to ensure perfect compatibility between the specific versions of React 18, React Three Fiber, and Drei).*

4. **Start the development server:**
   ```bash
   npm run dev
   ```

5. **Experience the Journey:** Open [http://localhost:3000](http://localhost:3000) in your browser and start scrolling!
